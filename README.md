# 🎮 3D Game with AI - MIKKU CN

Simple 3rd Person Game built with React + Three.js

---

## 🧩 Overview

This project is a simple 3D third-person game built using modern web technologies.  
It demonstrates how to create an interactive browser-based game with basic AI elements.

Built with:
- React
- Three.js
- React Three Fiber
- cannon-es (physics)

Three.js is a JavaScript library that enables rendering 3D graphics directly in the browser without low-level WebGL complexity. :contentReference[oaicite:0]{index=0}

---

## ✨ Features

- 3rd person camera system
- Character movement (WASD)
- Basic physics system
- Simple AI behavior
- Game loop & state handling
- Lightweight and extendable structure

---

## 📦 Getting Started

Follow these steps to run the project locally.

### 1. Clone the repository

```git clone https://github.com/Mikku44/3d-game-with-ai-mikkucn.git```

---

### 2. Navigate into the project folder

```cd 3d-game-with-ai-mikkucn```

---

### 3. Install dependencies

```npm install```

---

### 4. Start the development server

```npm start```

---

## 🖥️ Run the project

After running:

npm start

Open your browser and go to:

http://localhost:3000

---

## 🎮 Controls

W / A / S / D  - Move character  
Mouse          - Rotate camera  
E          - Interact / Start game  
Left-Click          - Interact / Start game  
Right-Click          - Interact / Start game
